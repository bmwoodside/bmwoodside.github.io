Completed a Coding Boot Camp!

I completed Coding Dojo's rigorous 14-week boot camp consisting of instruction in MERN (MongoDB, Express, React, Node), Python with Flask, and Java with Spring/Spring Boot! Classes were full time, Monday through Friday, 9a-5p, after which I'd spend time collaborating with peers; all-in-all I found myself dedicating more than 70 hours a week to absorb all of these languages!

Technologies I'm Experienced With!

Thanks to Coding Dojo, I've had quite a range of exposure to various technologies! Here's an abbreviated list of some of the most popular things: JavaScript (ES6) with React, Express, and Node; Python with Flask; Java with Spring Boot and Spring Security; Front-end development with HTML5, CSS3, Bootstrap, and Material-UI; Databases like MongoDB, MySQL, and PostgreSQL; Concepts such as Object-Oriented Programming, MVC architecture, and RESTful API's; plus additional tools like deployments using AWS and git (GitHub)!
